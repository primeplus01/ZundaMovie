・つかいかた
  拡張編集プラグインのあるディレクトリに、
  「script」というフォルダを作って、
  ごみ箱に解凍したものをフォルダごと入れます。
  詳しくは拡張編集のexedit.txt参照。

  ◆フォルダ構成例
    aviutl <dir>
    ├ aviutl.exe		：AviUtl本体
    └ plugins <dir>
       ├ exedit.auf		：拡張編集プラグイン（の一部）
       └ script <dir>
          └ 白水 <dir>
             ├ @hksy.anm	：アニメーション効果用のスクリプトファイル
             └ @hksy.obj	：カスタムオブジェクトのスクリプトファイル

    環境によっては、フォルダ構成に若干の違いがあると思います。
    重要なのは、exedit.aufと同じ階層にscriptフォルダがあることです。多分。


  それぞれのスクリプトの使い方は、
  トラックバーをいじって適当に考えてください。
